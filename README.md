## 运行

编译环境: Debian 11 bullseye

node 版本: v18 LTS

```bash

# 启用 pnpm
npm install -g pnpm

# 安装依赖
pnpm install

# 更新依赖
pnpm run update

# 运行
pnpm run dev

# 编译打包
pnpm run build

# 预览
pnpm run preview

# 更新同步
pnpm run sync

# 发布到 https://blog.mo7.cc
pnpm run deploy

```
