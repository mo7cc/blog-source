### 运行

编译环境: Debian 12

node 版本: v22.x

pnpm 版本: 9.x

```bash

# 启用 pnpm
npm install -g pnpm

# 安装依赖
pnpm install

# 更新依赖
pnpm run update-package

# 运行
pnpm run dev

# 编译打包
pnpm run build

# 更新同步
pnpm run sync

# 发布到 https://mo7.cc
pnpm run deploy

```
