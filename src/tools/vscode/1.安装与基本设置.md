# 1.安装与基本设置

## 安装流程

可以参考这篇文章：
[Visual Studio Code 入门教程](https://juejin.cn/post/7037449594900316173)

## 基本设置

初次使用建议点开左下角设置浏览一下设置项都有哪些。

右键侧边栏可以控制显示哪些选项。勾选账户。
然后登录你的 Github 或者 Microsoft 账户，这样就可以在多台设备中同步你的设置以及插件了。

![显示账户选项，然后登录Github](./image/login.png)
